# Shield

Built on [Double Ratchet Algorithm](https://en.wikipedia.org/wiki/Double_Ratchet_Algorithm).

The following cryptographic algorithms were selected: [X25519](https://en.wikipedia.org/wiki/Curve25519) for key exchange [HKDF-SHA256](https://en.wikipedia.org/wiki/HKDF) for root keys derivation and [HMAC-SHA256](https://en.wikipedia.org/wiki/HMAC-SHA256) for chain keys and message keys derivation.

See also [cipher](../cipher/)'s encryption algorithms.
